Class: Dialog

Dialog

new Dialog(text, width, height, options, callback)

Used to pause the game while waiting for user input. Uses the Canvas to show a HTML/CSS dialog box, returns the selection, closes, and resumes the game.
Parameters:
Name Type Description
text string The text to show in the dialog message (prompt).
width number The width of the dialog box.
height number Optional height of the dialog box. Use 0 to auto-size to the given text and options.
options Array.<string> List of options to choose from.
callback function Function to call when an option is selected, passing selected index.
